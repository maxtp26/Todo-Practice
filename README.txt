This project is a fairly simple todo list that saves and stores information in the file todos.json stored in the same folder.

Supported commands:
	-add [Item]
		-adds an item to the list at the last slot
	-remove [index]
		-removes the item on the indexth slot (starting from 1)
	-list
		-displays the entire list
	-done or check [index]
		-marks the indexth item as done
	-undo or uncheck [index]
		-marks the indexth item as undone
	-quit
		-exits the program
